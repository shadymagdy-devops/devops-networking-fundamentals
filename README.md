# DevOps Networking Fundamentals

A structured 7-day hands-on networking study plan for DevOps and Cloud Engineering.
All commands executed on a live Ubuntu 22.04 environment. All concepts applied directly to real DevOps workflows.

## Study Plan

| Day | Topic | Tools |
|-----|-------|-------|
| [Day 1](./day-01-foundation/) | IP Addresses, OSI Model, TCP/UDP | ip, ping, traceroute, ss, curl |
| [Day 2](./day-02-subnetting/) | Subnetting and CIDR Notation | ipcalc, manual subnet design |
| [Day 3](./day-03-dns/) | DNS — Records, Resolution, Internal | dig, nslookup, /etc/hosts |
| [Day 4](./day-04-firewalls/) | Routing, Firewalls and Security Groups | ufw, iptables, AWS Security Groups |
| [Day 5](./day-05-lb-nat-vpn/) | Load Balancers, NAT and VPN | AWS ALB, NLB, NAT Gateway |
| [Day 6](./day-06-http-tls/) | HTTP, TLS and Network Troubleshooting | curl, tcpdump, openssl, netcat |
| [Day 7](./day-07-aws-vpc/) | Full AWS VPC Architecture | Terraform, AWS Console, Bastion Host |
